# Installation
1) git clone into your PC

 
    git clone <url>

2) Activate virtual enviroment


    . ./venv/bin/activate

3) install requirements.txt 

 
    pip3 install -r requirements.txt

4) Run server on local host


    python3 manage.py runserver
