# Installation
1) git clone into your PC

 
    git clone <url>
 
2) Create virtual enviroment
 
    python3 -m venv venv

3) Activate virtual enviroment


    . ./venv/bin/activate

4) install requirements.txt 

 
    pip3 install -r requirements.txt

5) Run server on local host


    python3 manage.py runserver
